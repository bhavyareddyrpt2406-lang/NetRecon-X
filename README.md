# NetRecon-X

NetRecon-X is a Python-based Network Reconnaissance and Security Analysis Tool designed to identify open ports and analyze network services through an interactive web dashboard.

## Features

* TCP Port Scanning
* Service Detection
* Flask-Based Dashboard
* Interactive Web Interface
* Security Analysis Foundation
* Scan Result Visualization

## Technologies Used

* Python
* Flask
* Socket Programming
* HTML
* CSS

## Current Progress

* [x] Port Scanner
* [x] Flask Dashboard
* [x] Scan Results Integration
* [ ] Scan History
* [ ] SQLite Database
* [ ] Network Discovery
* [ ] Risk Scoring
* [ ] AI Anomaly Detection

## Project Goal

Build a complete Network Reconnaissance and Security Analysis Platform capable of device discovery, threat assessment, anomaly detection, and reporting.
